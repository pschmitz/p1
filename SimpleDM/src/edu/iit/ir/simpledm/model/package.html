Provides classes for building and testing a model.

Model class is a super class of all models. Four abstract methods must be implemented to extend the Model class.
Only SVM model is implemented for Verson 1.1. A student will create other assigned models extending the abstract Model class.

A SampleSet is passed as a parameter to be preprocessed. The preprocessed SampleSet will be passed for building and testing a model.
