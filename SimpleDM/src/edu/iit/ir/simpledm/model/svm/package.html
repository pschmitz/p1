Provides classes for implementing Support Vector Machine. It extends an abstract Model class.

Package Specification

  • Building a Model : Creates an independant model for each training SampleSet. SVM saves the models including support vectors and its alphas.
  • Testing Models : Applies a testing SampleSet to every built models, and votes the majority of results.
  • Kernel Functions: A Polynomial and Radio Basis kernel functions were implemented.
  • SVM_ParamFrame: Extends ParamFrame so that a user test different parameters.
  • Default parameters are set. These defaults are decided after some testing with 'adult.arff' file and 3-fold cross validation. With another data file and N-folds, the parameters should be adjusted by a user using SVM_ParamFrame.
