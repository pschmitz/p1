Provides classes for preprocessing a SampleSet and general information of the data set.

Package Specification

  • Preprocessor : Contains preprocessing services and boolean values of each service indicating if the preprocessing is selected by a user or not.
  • DataSet : Contains general informaiton such as relation name and attributes statistics and label distribution.
    It is used by Preprocessor and a model. For example, minimum, maximum, average and standard deviation of each continuous attributes are used to normalize a SampleSet in a Preprocessor.normalizeMinmax(SampleSet set) function.

    Note: A DataSet contains the general information, while a SampleSet contains the actual records to be preprocessed.
  • DataDiscretizer : Transforms from continuous type to cateogrical. A student can extend the DataDiscretizer class to implement more sophiscated discretizer.
