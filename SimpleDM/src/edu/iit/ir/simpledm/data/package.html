Provides classes for storing a block of actual data to preprocess, and build and test a model with.
Containing a block of data not the whole records in a file, is because of the scalability, which means that it is assumed that the data file is too big to fit in memory.
Becaue of this scalability, {@link Model} reads a block of data from the file , and stores it in memory with a SampleSet-format. Then it sends the {@link SampleSet} to {@link Preprocessor} and Model.buildModel(SampleSet set) and Model.testModel(SampleSet set) as a parameter.

Package Specification

  • A SampleSet contains a chunk of Samples(records).
  • A Sample contains one instance(record), that is,an array of attributes including class.
  • An AttributeValue has the type of the attribute and either CaegoricalAttributeValue or ContinuousAttributeValue object.

Here is an example to use those classes.

void exampleSampleSet(SampleSet set)
{
Sample[] samples = set.getSamples();
// the first attribute of the first record
Attribute attr = samples[0].get(0);
// if the type of the attribute is categorical
if(attr.getType() == Sample.TYPE_CATEGORICAL)
{
String label = ((CategoricalAttributeValue)attr).getValue();
// do something..
}
....
}
